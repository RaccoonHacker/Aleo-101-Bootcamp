# Control Flow

## Conditional Statements（条件语句）

Conditional statements（条件语句）使用 `if {condition} {...} else if {condition} {...} else {...}` 的形式声明。

条件语句可以嵌套使用。

```leo
let a: u8 = 1u8;

if a == 1u8 {
    a += 1u8;
} else if a == 2u8 {
    a += 2u8;
} else {
    a += 3u8;
}
```

Leo 也支持 ternary expressions（三元表达式）。三元表达式使用 `{condition} ? {then} : {else}` 的形式声明，并且也可以嵌套使用。

```leo
let a: u8 = 1u8;
a = (a == 1u8) ? a + 1u8 : ((a == 2u8) ? a + 2u8 : a + 3u8);
```

## For Loops（For 循环）

For loops（For 循环）使用 `for {variable: type} in {lower bound}..{upper bound}` 的形式声明。

循环边界必须是相同类型的 integer constants（整数常量）。此外，如果 lower bound（下界）大于或等于 upper bound（上界），该循环将不会执行任何操作。

Leo 支持 nested loops（嵌套循环）。

```leo
let count: u32 = 0u32;

for i: u32 in 0u32..5u32 {
    count += 1u32;
}

return count; // returns 5u32
```

## Return Statements（返回语句）

Return statements（返回语句）使用 `return {expression};` 的形式声明。

```leo
let a: u8 = 1u8;

if a == 1u8 {
    return a + 1u8;
} else if a == 2u8 {
    return a + 2u8;
} else {
    return a + 3u8;
}
```
